This is an example web application to display my abilities as a developer. 

It is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app). It uses Typescript, ESLint, and Tailwind CSS.

## Installation

Clone the repo

cd into the repo directory

```bash
yarn install
```

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
